## API Documentation

- [BASE_URL](https://hidden-bayou-75178.herokuapp.com/)

Get /api/bot

- Provides array of bot name

Example request/response:

```
GET https://hidden-bayou-75178.herokuapp.com/api/bot

HTTP STATUS 200 OK

[
 "evi"
]

```

POST /api/bot

Example request/response:

```

POST https://sleepy-harbor-21485.herokuapp.com/api/bot
REQ BODY: {
    "bot_name": "evi",
}

Response:
[
"Evi"
]

```
